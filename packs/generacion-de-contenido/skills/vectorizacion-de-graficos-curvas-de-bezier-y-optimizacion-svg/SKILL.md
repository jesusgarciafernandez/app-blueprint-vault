## 0. Filosofía Human-Centric AI
*Esta habilidad destila la forma pura en matemáticas invisibles, utilizando la tecnología para crear gráficos eternos, ligeros y perfectos en cualquier escala.*

**El Rol del Humano:** El vectorista debe ser un "Geómetra de la Belleza". La IA puede trazar automáticamente una imagen de píxeles y sugerir formas simplificadas, pero solo el humano puede asegurar que el trazado tenga la elegancia visual de una curva perfecta, decidir qué puntos de ancla son necesarios para mantener la fluidez del diseño y garantizar que el gráfico final transmita la autoridad técnica y el rigor estético que la marca requiere.
**Empoderamiento:** Usamos la tecnología para automatizar la limpieza de nodos y la generación de variaciones geométricas, permitiendo que el diseñador se centre en la síntesis conceptual y en la creación de sistemas visuales infinitamente escalables y ultra-ligeros.

## 1. Descripción Detallada
La Vectorización de Gráficos profesional es la creación de activos visuales basados en vectores (fórmulas matemáticas). No es solo "dibujar con la pluma"; es **Ingeniería de la Forma Escalable**. El enfoque v2.0 incorpora la **Simplificación de Nodos Inteligente y la Optimización SVG de Alto Rendimiento**, donde cada gráfico (Logo, Icono, Ilustración) se construye con el número mínimo de puntos de ancla para asegurar una ligereza extrema en la web (SEO técnico) y una nitidez absoluta en pantallas de cualquier densidad (Retina/4K) o formatos de impresión gigantescos.

## 2. Escenarios de Aplicación
- **Digitalización de Logotipos y Símbolos de Marca:** Conversión de bocetos a mano en archivos maestros (.AI/.SVG) listos para cualquier uso.
- **Creación de Sistemas de Iconografía a Medida:** Diseño de iconos para Apps que pesen poco y se vean perfectos en cualquier resolución.
- **Ilustración Técnica y Planometría:** Creación de diagramas precisos para manuales de instrucciones o planos arquitectónicos.
- **Preparación de Activos para Producción Física:** Trazados de corte precisos para vinilos, grabado láser, fresado CNC o bordados textiles.
- **Gráficos para Animación Web (Lottie/GSAP):** Diseño de capas vectoriales perfectamente organizadas para ser animadas por código sin penalizar el rendimiento.

## 3. Requisitos de Implementación
- **Maestría en Herramientas de Dibujo Vectorial:** Adobe Illustrator (estándar), Affinity Designer o Inkscape (open source).
- **Control Milimétrico de Curvas de Bézier:** Capacidad de manejar manejadores y tangentes para crear formas fluidas y simétricas.
- **Entendimiento de Formatos y Rendimiento:** Conocimiento de cómo limpiar el código interno de un archivo SVG para minimizar KB innecesarios.

## 4. Diferencial: Calco Automático vs. Vectorización de Precisión v2.0

| Dimensión | Enfoque Casual (v1.0) | Vectorización Profesional (v2.0) |
| :--- | :--- | :--- |
| **Calidad de Curva** | Dentada / "Sucia" (Muchos nodos). | Fluida / Matemática (Nodos mínimos). |
| **Escalabilidad** | Errores en zoom extremo. | Perfección absoluta a cualquier tamaño. |
| **Peso del Archivo** | Pesado e ineficiente. | Ultra-ligero (Optimizado para web). |
| **Editabilidad** | Difícil / Caótica. | Estructurada por capas y grupos lógicos. |

## 5. Instrucciones y Pasos Detallados (Protocolo Maestro)

### Fase 1: Calco de Base y Estructuración de Trazados Principal
**Objetivo:** Capturar la esencia de la forma original con la máxima fidelidad técnica.
1.  **Prep del Imagen de Base:** Sube el contraste del boceto original para facilitar el calcado.
2.  **Trazado de Puntos de Ancla:** Sitúa los nodos en los puntos de inflexión (extremos de las curvas) para un control total mediante el número mínimo de puntos.

**Prompt Maestro de Dirección de Vectorización:**
```text
Actúa como Especialista en Diseño Vectorial y Operador Senior de Illustrator. Para la vectorización del [ACTIVO] de la marca [MARCA], realiza el siguiente plan técnico: 
1. Define el método de trazado: [Ej: Trazado manual con pluma para máxima precisión / Calco de imagen con refinado posterior]. 
2. Especifica la jerarquía de capas: [Ej: Capa de Contornos / Capa de Color / Capa de Detalles]. 
3. Indica el protocolo de simplificación: Uso de la herramienta 'Simplificar' para reducir nodos manteniendo la integridad de la forma original. 
4. Describe el acabado: [Ej: Formas planas sin bordes / Uso de mallas de degradado para realismo / Línea de grosor variable]. 
5. Detalla el formato de exportación: [SVG optimizado con 2 decimales de precisión para web / PDF-X para producción física].
```
