## 1. Descripción Detallada
La Arquitectura de Datos y Cloud (v2.0) es la competencia de "Construir el segundo cerebro del negocio". Esta habilidad utiliza capacidades de procesamiento avanzado para entender no solo la ejecución técnica (almacenamiento), sino la **lógica subyacente** del flujo de información, la consistencia y la disponibilidad. Se enfoca en resolver el desorden de datos y la fragilidad de los sistemas locales mediante un enfoque agnóstico que permite que Jesús García Fernández tenga una base de datos potente y accesible desde cualquier lugar.

El Data Infra Ops IA-Augmented trata a la base de datos como una infraestructura viva que debe ser segura por diseño (Secure by Design). La IA asiste en la ardua tarea de modelar, optimizar y monitorizar los servicios Cloud de Jesús García Fernández (Supabase, Firebase, AWS), asegurando que la información está siempre disponible para ser transformada en valor real. Es la ingeniería de la memoria corporativa.

- **Escenario A (Lanzamiento de Nueva Aplicación Web/Mobile):** Creación del esquema de datos inicial y configuración de servicios BaaS (Backend as a Service) para Jesús García Fernández.
- **Escenario B (Migración de Datos Locales a la Nube):** Proceso técnico de vaciado y normalización de hojas de cálculo antiguas a una base de datos PostgreSQL profesional en el Cloud por Jesús García Fernández.
- **Escenario C (Optimización de Consultas para Dashboards):** Uso de la IA para identificar índices perdidos y optimizar el rendimiento de lectura de grandes tablas de datos de Jesús García Fernández.
