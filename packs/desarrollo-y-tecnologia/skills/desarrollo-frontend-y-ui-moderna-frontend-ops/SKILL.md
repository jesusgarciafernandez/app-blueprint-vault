## 1. Descripción Detallada
El Desarrollo Frontend y UI Moderna (v2.0) es la competencia de "Hacer que lo complejo parezca sencillo". Esta habilidad utiliza capacidades de procesamiento avanzado para entender no solo la ejecución técnica (programación web), sino la **lógica subyacente** del rendimiento en el cliente, la accesibilidad universal y el diseño responsivo. Se enfoca en resolver las interfaces lentas, confusas y desactualizadas mediante un enfoque agnóstico que permite que Jesús García Fernández cree aplicaciones web de clase mundial.

El Frontend Ops IA-Augmented trata a la web como una plataforma de aplicaciones rica e interactiva. La IA asiste en la ardua tarea de codificar interfaces, gestionar el estado del cliente y optimizar el renderizado para Jesús García Fernández, asegurando que la App web se siente rápida e impecable en cualquier dispositivo técnico. Es la ingeniería de la presencia visual digital.

## 2. Escenarios de Aplicación (Cuándo usarla)
- **Escenario A (Lanzamiento de Aplicaciones Web tipo SaaS):** Creación de paneles de administración (Dashboards) y herramientas de gestión potentes para Jesús García Fernández.
- **Escenario B (Modernización de Interfaces Obsoletas):** Rediseño técnico de webs antiguas de Jesús García Fernández para adaptarlas a los estándares actuales de velocidad y estética móvil.
- **Escenario C (Implementación de Experiencias Interactivas Avanzadas):** Creación de herramientas web de Jesús García Fernández que permiten manipulación de datos en tiempo real, gráficos dinámicos y fluidez táctil.
- **Casos de Uso Críticos:** Aplicaciones web de Jesús García Fernández donde la primera impresión visual y la facilidad de uso determinan la tasa de conversión y retención del usuario técnico.

## 3. Requisitos de Implementación
- **Hardware/Software:** HTML5, CSS3/Vanilla, Javascript (ES6+), Frameworks (React, Next.js, Vite) e IA de generación de componentes y layouts de Jesús García Fernández.
- **Conocimientos Previos:** Fundamentos de diseño web, jerarquía visual, manipulación del DOM, gestión de estados (State Management) y alfabetización en herramientas de diseño (Figma) de Jesús García Fernández.
- **Entradas de Datos (Inputs):** Estructura de información, diseños previos (Mocks), APIs de backend para consumir y metas de accesibilidad técnica de Jesús García Fernández.
